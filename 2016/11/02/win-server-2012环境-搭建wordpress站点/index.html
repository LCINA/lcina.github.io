[win server 2012环境] 搭建wordpress站点

发表于 2016-11-02 | 阅读次数:

今天搭了一下wordpress站点，之前是自己搭建过的，可是好多步骤都给忘了，所以这次又踩了好多坑，这次就记录一下过程：

  1. 首先肯定要购买一台云服务器，是学生的话直接一元购；
  2. 【云指南】腾讯云服务器环境配置与解决方案指南 ,配置可以直接参考官方论坛的；配置IIS，直接参考官方的教程就行；接下来是Mysql环境的配置；最后是php环境配置，按照官方教程来，不过最后一步的PHP拓展板块开启，那会儿我全都没开，后面就入坑了；
  3. 下载wordpress、phpMyAdmin，在上面安装好iis后，进入C:\inetpub\wwwroot；新建一个文件夹phpMyAdmin（名字自己取）,将所有文件复制到该文件夹，保证php配置好的情况下，打开浏览器，输入 http://localhost/phpMyAdmin，回车，如果上面PHP环境配置中的拓展板块未开启，会出现500错误，所以这问题搞了有一会儿，最后一怒之下把所有PHP拓展板块全开了，就是把前面的分号去掉就行，不过好像只需要开这个就行——extension=php_mbstring.dll；保存php.ini文件后，重新刷新浏览器，登录界面出来了，用之前配置的Mysql的账号密码进行登录，登录进去后，新建数据库，命名自己喜欢就好；
  4. 将下载后的wordpress文件解压，复制所有文件丢到C:\inetpub\wwwroot；打开浏览器输入http://localhost/，出现wordpress配置界面，这里的步骤就很简单了，可以参考Wordpress程式搭建教程（Windows环境） ，最后就搞定了
